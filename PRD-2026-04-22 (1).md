

# GharSoch — AI-Powered Real Estate Financial Advisory & Voice Call Centre Platform

[SELECTED_THEME: luxury-dark]

## 1. Overview
GharSoch is an AI-powered real estate financial advisory platform that helps homebuyers make confident property purchase decisions. At its core, the platform provides a comprehensive affordability assessment engine — calculating total property cost (base price, GST, stamp duty, registration, PLC), mapping it against the buyer's complete financial profile (income, existing EMIs, expenses), and delivering a clear **Go / Reconsider / No-Go** affordability signal with tranche-wise cash outflow analysis. Layered on top is an autonomous voice call centre powered by RealVoice AI agents that handle inbound lead inquiries, outbound post-showing follow-ups, and proactive property re-engagement — collecting financial parameters conversationally, running GharSoch affordability analysis in real-time during calls, navigating buyer objections, searching live property listings, and booking meetings. Every interaction feeds back into a unified brokerage dashboard with sentiment analytics, affordability insights, and a self-service GharSoch tool for manual assessments with a paid detailed report option (₹999).

## 2. User Stories
- As a homebuyer, I want to input a property's full cost breakdown and my financial profile so I get a clear Go/Reconsider/No-Go affordability signal before committing.
- As a brokerage manager, I want the AI voice agent to collect a buyer's financial details during a follow-up call and run GharSoch affordability analysis in real-time so hesitant buyers get instant guidance.
- As a brokerage manager, I want autonomous voice agents handling inbound inquiries, property searches, objection navigation, and meeting scheduling so no lead goes cold.
- As a brokerage manager, I want clients onboarded with T&C consent so the system proactively re-engages them when matching affordable properties arise.
- As a brokerage manager, I want a dashboard showing call analytics, affordability signal distribution, and conversion funnels so I can monitor performance and coach agents.

## 3.a. Agent Architecture

**Pattern:** Manager-Subagent + Independent Agents (Hybrid)

**Reasoning:** Live voice calls require a single trigger orchestrating multiple specialized LLM tasks in one shot — lead qualification, GharSoch financial advisory, property RAG search, and calendar scheduling. These are distinct specializations needing separate tools and contexts. Three Independent Agents handle post-call sync (natural breakpoint after call ends), daily property re-engagement (scheduled), and the self-service GharSoch affordability tool (user-triggered from dashboard).

**Agent Flow:**
When a CRM webhook fires or an inbound call arrives, the **Voice Conversation Orchestrator (Manager)** is triggered and coordinates the live call. It routes intent to: **Lead Qualification & Objection Agent** for preference gathering, objection handling (spousal, pricing, trust), and T&C consent capture. **GharSoch Financial Advisory Agent** activates on financial hesitation — collects income, EMIs, expenses conversationally, calculates total property cost (itemised: base + parking + club + PLC + GST + stamp duty + registration), computes EMI/pre-EMI, runs affordability signal logic (excess ratio → Go/Reconsider/No-Go), discusses tranche-wise outflow, and recommends financial advisor follow-up if needed. **Property Search Agent** queries MLS knowledge base via RAG for property details and budget-matched alternatives. **Calendar Scheduling Agent** checks Google Calendar availability and books meetings.

After call termination, **Post-Call Sync Agent** (Independent) processes the transcript — extracts sentiment, objections, affordability signal, financial readiness — and stores results. **Property Re-engagement Agent** (Independent, daily schedule) matches new listings against stored client preferences and queues outbound calls pending broker approval. **GharSoch Self-Service Advisor** (Independent) powers the dashboard affordability tool where staff input property costs and financial profiles via a4-step wizard, receive the full GharSoch analysis, and generate the₹999 paid PDF report.

## 4. User Flow

1. User lands on GharSoch platform (visible but locked) with auth modal overlay — Sign Up default, luxury-dark theme.
2. New user → Sign Up (name, email, password) → account created → dashboard. Returning user → Log In → dashboard.
3. **Dashboard:** KPI cards (Active Calls, Calls Today, Avg Sentiment, Meetings Booked, Go/Reconsider/No-Go distribution). Live Call Activity feed with sentiment and affordability signal badges. Quick Call Log table. Escalation alert banner with warm transfer action.
4. **Voice Call (Inbound/Outbound):** Voice Conversation Orchestrator auto-handles → routes to Lead Qualification (preferences, objections, T&C) → GharSoch Financial Advisory (collects financial profile conversationally, calculates property affordability, delivers signal with tranche-wise guidance, schedules advisor if needed) → Property Search (RAG for details/alternatives within validated budget) → Calendar Scheduling (books meeting). Escalation → warm transfer + alert.
5. **GharSoch Tool (Self-Service):** User navigates to \"GharSoch\" → Step 1: Property Cost entry (lump sum or itemised with auto-calculated total) → Step 2: Payment Plan (UC/RTM, loan config, tranche builder with running total to100%) → Step 3: Financial Profile (income sources, existing obligations, auto-calculated total outflow) → Step 4: \"Analyze Affordability\" CTA triggers GharSoch Self-Service Advisor → Advisory Output displayed (signal badge, tranche-wise breakdown table with colour-coded status per tranche, income vs outflow bar chart, tier-specific recommendations). → \"Get Full Report — ₹999\" CTA → payment → PDF download.
6. **Campaigns:** Outbound queue (CRM leads) + Re-engagement alerts (new property matches pending approval). Approve/trigger calls.
7. **Call Logs:** Filterable table → Call Detail Modal (transcript, sentiment timeline, financial parameters, affordability signal, tranche analysis, meeting booked). \"Sync & Analyze\" CTA.
8. **Analytics:** Call volume, sentiment distribution, conversion funnel (calls → qualified → financial consult → meeting), affordability signal distribution, re-engagement success rates.
9. **Settings (Admin):** Agent persona, escalation rules, voicemail/SMS config, campaign settings, GharSoch defaults (GST%, stamp duty suggestions, affordability thresholds, disclaimer), user management.
